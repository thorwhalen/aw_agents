aw_agents.agents.download.download_core

Download Agent: Smart content downloader with context-aware naming and intelligent link handling.

This agent can download content from URLs with special handling for: - Indirect download links (landing pages with actual download links) - Scientific papers (PDFs from various sources) - Data from HuggingFace, Kaggle, GitHub - Context-aware file naming based on conversation context
