aw_agents.adapters.openapi

OpenAPI/FastAPI adapter for AW agents.

This adapter wraps any AgentBase subclass to expose it as a REST API with OpenAPI spec for use with ChatGPT Custom GPTs.

class aw_agents.adapters.openapi.OpenAPIAdapter(agent: AgentBase, *, title: str = 'AI Agent API', description: str = 'AI Agent exposed via OpenAPI', version: str = '1.0.0', server_url: str | None = None)[source]

Adapter to expose an AgentBase as a FastAPI/OpenAPI service.

Usage:

agent = YourAgent() adapter = OpenAPIAdapter(agent, title=”Your Agent API”) adapter.run(port=8000)
