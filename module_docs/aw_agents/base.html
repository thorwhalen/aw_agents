aw_agents.base

Base classes and protocols for AW agents.

This module provides the foundation for building AI agents that can be deployed to multiple chatbot platforms (Claude, ChatGPT, etc.).

class aw_agents.base.AgentBase[source]

Base class for all AW agents.

Agents should inherit from this and implement the core methods. Adapters (MCP, OpenAPI) will wrap these agents for deployment.
